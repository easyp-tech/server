### 2. Rewrite `newLogger()` ‚Äî Accept full `LogConfig`, support format + source + env overrides

**File:** `cmd/easyp/main.go`

**Changes:**
- Change signature from `newLogger(level string)` to `newLogger(cfg config.LogConfig)`
- Add `EASYP_LOG_LEVEL` env var override with precedence over config
- Add `EASYP_LOG_FORMAT` env var override over config `Format` field
- Switch on `Format` to create `slog.NewJSONHandler` or `slog.NewTextHandler`
- Wire `AddSource` into `slog.HandlerOptions.AddSource`
- Implement `slog.HandlerOptions.ReplaceAttr` for centralized sensitive-data redaction

**`ReplaceAttr` behavior:**
- Inspect `slog.Attr.Key` for keys matching sensitive patterns (`token`, `password`, `secret`, `authorization`, `access_token`, `api_key`, `auth`)
- Replace matching values with `"***"`
- Leave `time`, `level`, `source`, `msg` keys untouched
- Must be applied at the handler level so it covers ALL log output ‚Äî not just middleware-level logging

```go
func newLogger(cfg config.LogConfig) *slog.Logger {
	// Env var override
	levelStr := cfg.Level
	if envLevel := os.Getenv("EASYP_LOG_LEVEL"); envLevel != "" {
		levelStr = envLevel
	}

	var logLevel slog.Level
	switch strings.ToLower(levelStr) {
	case "debug":
		logLevel = slog.LevelDebug
	case "warn", "warning":
		logLevel = slog.LevelWarn
	case "error":
		logLevel = slog.LevelError
	default:
		logLevel = slog.LevelInfo
	}

	formatStr := cfg.Format
	if envFormat := os.Getenv("EASYP_LOG_FORMAT"); envFormat != "" {
		formatStr = envFormat
	}

	opts := &slog.HandlerOptions{
		Level:     logLevel,
		AddSource: cfg.AddSource,
		ReplaceAttr: func(groups []string, a slog.Attr) slog.Attr {
			if isSensitiveAttr(a.Key) {
				return slog.String(a.Key, "***")
			}
			return a
		},
	}

	var handler slog.Handler
	switch strings.ToLower(formatStr) {
	case "text":
		handler = slog.NewTextHandler(os.Stdout, opts)
	default:
		handler = slog.NewJSONHandler(os.Stdout, opts)
	}

	return slog.New(handler)
}
```

**Sensitive attribute matching function (add `isSensitiveAttr` to `main.go`):**

```go
func isSensitiveAttr(key string) bool {
	k := strings.ToLower(key)
	return k == "token" || k == "password" || k == "secret" ||
		k == "authorization" || k == "access_token" ||
		k == "api_key" || k == "auth"
}
```
